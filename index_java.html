Introducción al lenguaje Java






Descripción: El material que se presentará para este curso serán únicamente apuntes. El curso busca dar las nociones básicas del lenguaje Java, desde los conceptos más básicos hasta la programación orientada a objetos. Como requisito para llevar este curso es imprescindible tener conocimientos básicos de programación, así como nociones sobre la poo. Este curso está diseñado para ser llevado por el estudiante que ya tiene conocimientos en programación, especialmente en python, así como conocimientos en poo, de nuevo, especialmente en python.
