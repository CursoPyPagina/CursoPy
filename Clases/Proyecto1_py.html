Grid manager ¶

Podemos considerar a nuestra ventana como un plano coordenado, de modo que las posiciones de los componentes, as como lo vimos antes, estarn posicionados mediante coordenadas. Notamos que la manera en que estuvimos colocando las coordenadas de nuestros componentes realmente no era la ptima, pues si quisieramos colocar muchos otros ms componentes tendramos que estar jugando con las coordenadas para colocar todos estos en la posicin que deseamos. Una manera ms precisa de utilizar las coordenadas es mediante el mtodo grid(row=n, column=m) que divide toda nuestra ventana en filas (n) y columnas (m). Por ejemplo, podemos llegar a considerar nuestra ventana como una cuadrcula de 4 filas por 4 columnas y un sistema coordenado como se muestra a continuacin

Teoría ¶

La idea intuitiva de lmite coincide con la idea de aproximacin. Por ejemplo, cuando intentamos calcular $\lim\limits_{x\rightarrow 3}x^{2}$, intentamos hallar el valor prximo (no igual) de la funcin $f(x)=x^{2}$ cuando $x$ se aproxima o tiende al valor de 3. Con lo anterior podemos realizar algunas aproximaciones para obtener el valor de dicho lmite:

Bsicamente esa es la idea detrs de los lmites, pero el objetivo de esta clase ser resolver lmites utilizando cierto cdigo en Python mediante la librera SymPy.

Por otra parte, podemos recordar que, para el clculo de derivadas, podamos basarnos en frmulas. Por ejemplo, tenamos la siguiente frmula

$$ \frac{d}{dx}x^{n} = n x^{n-1}, \ \ \ \ \ n\neq 0 $$

Si consideramos la funcin $f(x)=x^{2}$, tenemos entonces, utilizando la frmula anterior, que

$$ \frac{d}{dx}x^{2} = 2x^{2-1}=2x $$

Símbolos matemáticos ¶

Recordemos que en Python podemos asignarle cualquier nombre a nuestras variables, siguiendo ciertas normas. As que, sin problema alguno, podemos definir variables con los nombres x, y, etctera. Notemos que todas las variables que hemos ocupado en Python tienen un valor dado, un valor que previamente, de antemano, conocemos. Asimismo, dentro de matemticas est la nocin de variable, digamos $x$, la cual, en primera instancia, puede representar cualquier nmero y que de manera certera desconocemos. Por ejemplo, si definimos una variable x en Python, entonces operaciones como

2 * x + 5 * x - 2

seguramente nos darn un nmero fijo; si por ejemplo x tiene asignado el valor de 0, entonces la operacin anterior nos dar como resultado el nmero fijo 0; si cambiamos el valor de x, la operacin anterior nos dar un nmero fijo distinto. No obstante, en todo momento x debe tener asignado un valor, pues de lo contrario Python nos arrojar un error. De tal manera, con lo anterior no podemos "jugar" con la idea de que x represente a un nmero cualquiera.

Por ejemplo, recordemos de nuestros cursos bsicos de lgebra que:

$$ 2\cdot x + 5\cdot x - 2 = 7\cdot x - 2 $$

es decir, podemos simplificar la expresin algebraica sumando los trminos semejantes, donde, en primera instancia, no conocemos el valor de la $x$ pues sta representa la nocin de un nmero cualquiera, sin embargo, an as podemos realizar operaciones como la anterior.

Notamos entonces las diferencias entre una variable en Python (la cual siempre tiene un valor asignado) y las variables matemticas (las cuales pueden representar la idea de un nmero cualesquiera). De tal manera, para que Python logre diferenciar estos dos tipos de variables, especificaremos a las variables matemticas como smbolos matemticos.

Ahora, simplificaremos la expresin $2\cdot x + 5\cdot x - 2$ utilizando Python. Ahora, para calcular derivadas utilizaremos el mtodo diff(), donde en el primer parmetro colocamos la funcin a derivar y en el segundo parmetro respecto a qu variable estamos derivando. Por ejemplo, si queremos derivar la funcin $f(x)=x^{2}$ respecto a la variable $x$ ($\frac{d}{dx}x^{2}$) escribiremos entonces

In [47]:

# Definimos nuestra funcion matematica
# o simbolica

# calculo de la derivada

# vemos la derivada calculada:
derivada
Out[47]:
$\displaystyle 2 x$

de donde $\frac{d}{dx}x^{2}=2x$. Nota que en este caso escribimos 'x^2' en vez de simplemente x ** 2, las cuales son equivalentes.
