Librería yfinance ¶

Esta librera extrae informacin actualizada de yahoo finance referente a los principales mercados financieros en el mundo. As, podemos extraer informacin sobre precios de acciones de diversas compaias. Para acceder a la informacin de google utilizares el smbolo 'goog'; para el caso de Apple el smbolo es 'aapl'. Puedes chacar ms ndices en el siguiente Link.

Para extraer informacin deberemos de utilizar primero el mtodo Ticker(), donde dentro del parntesis colocaremos la cadena de texto con el smbolo de nuestro inters. Por ejemplo, para google tenemos:

In [31]:
# importacin necesaria
import yfinance as yf

Luego, para obtener los precios histricos de las acciones de dicha compaia utilizamos el mtodo history() al cual le pasaremos 3 parmetros:

  • interval: cadena de texto que especifica el perodo de tiempo en el cual extraeremos los datos. Por ejemplo, podemos especificar un perodo diario (es decir que obtenemos los precios de la accin de cada da), a un trimestre, a un semestre, entre otros.
  • start: cadena de texto con la fecha inicial a la cual extraeremos la informacin, donde el formato de la fecha es ao-mes-da.
  • end: fecha de fin de la extraccin de la informacin.

# importaciones necesarias
from keras.models import Sequential
from keras.layers import Dense
from sklearn.metrics import mean_squared_error
from sklearn.metrics import r2_score
import pandas as pd

def modelo(n1, n2, activacion, epocas, boolean):
    """
    n1: nmero de neuronas en la primer capa
    n2: nmero de neuronas en la segunda capa
    activacion: funcin de activacin en la ltima capa
    epocas: nmero de pocas
    boolean: booleano para determinar el valor de verbose
    """
    # Instanciamos el modelo
    model = Sequential()
    # Capa de entrada de 1 neurona y capa oculta de n1 neuronas.
    # Para la capa oculta utilizamos la funcion de activacion relu
    model.add(Dense(n1, activation='relu', input_shape=(3,)))
    # Capa oculta 2:
    # n2 neuronas y funcion de activacion relu
    model.add(Dense(n2, activation='relu'))
    # Capa de salida con funcion de activacion "activacion"
    model.add(Dense(1, activation=activacion))
    # Compilacion. Utilizaremos ahora el error cuadratico medio
    model.compile(optimizer='adam', loss='mean_squared_error')
    # Entrenamiento
    modelo = model.fit(X_train, y_train, epochs=epocas, verbose = boolean)
    # prediccion
    y_pred = model.predict(X_test)
    # dataframe con la informacion
    df = pd.DataFrame(y_pred).rename(columns = {0: 'y_pred'})
    df['y_test'] = y_test
    df = df.reset_index()
    df['diff'] = abs(df.y_pred - df.y_test)
    df['error'] = (df['diff'] * 100) / df['y_test']
    # estadisticas
    df_stats = df.describe()
    # grafica
    plt.plot(df.index, df.y_pred, color='blue', label='prediccion')
    plt.plot(df.index, df.y_test, color='red', label='real')
    plt.legend()
    plt.grid()
    plt.show()
    # MSE
    mse = mean_squared_error(y_pred, y_test)
    print(f'mse = {mse}')
    # hacemos que la funcion retorne 4 valores
    #                           |modelo de la red neuronal
    return modelo, df, df_stats, model
