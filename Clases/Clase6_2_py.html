Introducción ¶

Inicialmente, podemos entender como recursin al proceso en que una funcin se llama a s misma. Por ejemplo, lo dicho anteriormente se puede ilustrar con la siguiente idea:

def func(. . .):
    # Instrucciones
    # .
    # .
    func(. . .)
    # .
    # .

donde, dentro de las instrucciones de la funcin func, estamos utilizando propiamente la funcin func(), esto es, dentro de las instrucciones de la definicin de dicha funcin, estamos mandando a llamar a esa misma funcin. Tal vez por ahora parezca algo confunso, pero a lo largo de esta clase indagaremos ms a detalle sobre qu es la recursin y cmo la podemos implementar en Python.

Respecto a la recursin podemos decir que:

  • Es un proceso que se define en trminos de s mismo.
  • Se ocupa para dividir un problema en subproblemas que estn relacionados entre s, de modo que las soluciones a estos subproblemas, de manera global, representan la solucin del problema principal.
  • Podemos pensar a la recursin, al menos de manera intuitiva, como una "especie" de bucle.

Factorial de un número ¶

En matemticas, definimos el factorial de un nmero $n\in \mathbb{N}$, como el nmero obtenido de calcular $n\cdot (n-1)\cdot (n-2)\cdots 2\cdot 1$, a este nmero lo solemos denotar por $n!$. Por ejemplo:

  • $2! = 2\cdot 1 =2$
  • $3! = 3\cdot 2\cdot 1=6$
  • $5! = 5\cdot 4\cdot 3\cdot 2\cdot 1=120$

En otras palabras, podemos decir que el factorial de un nmero $n$, es la multiplicacin de $n$ con todos sus antecesores hasta llegar al nmero 1. Adems, podemos ver que el comportamiento anterior es muy similar al comportamiento de la funcin recursiva que definimos antes: func(). De tal manera, implementaremos una funcin recursiva para calcular el factorial de un nmero dado $n$, siguiendo la siguiente idea:

Una vez ingresado el valor de $n$, podemos calcular el factorial de $n$ como sigue: $$ n!=n\cdot (n-1)\cdot (n-2)\cdots 2\cdot 1=n\cdot (n-1)! $$ Luego, si queremos hallar el valor de $(n-1)!$ calculamos: $$ (n-1)!=(n-1)\cdot (n-2)! $$ y as sucesivamente hasta que llegamos a calcular $2!=2$, es decir, continuaremos calculando factoriales hasta que lleguemos a que dicho clculo ya es un nmero fijo, o a que nuestro clculo ya no involucra el smbolo $!$. Luego, dado que $2!=2$, regresamos para calcular los factoriales que quedaron pendientes:

  • $3!=3\cdot 2!=3\cdot 2=6$ y regresamos al factorial anterior
  • $4!=4\cdot 3!=4\cdot 6=24$ y regresamos al factorial anterior
  • $5!=5\cdot 4!=5\cdot 24=120$ y regresamos al factorial anterior $$ \vdots $$
  • $(n-1)!=(n-1)\cdot (n-2)!$, donde, para este punto, el valor de $(n-2)!$ ya es conocido, de modo que el nmero $(n-1)!$ tambin ya lo podemos conocer. Finalmente regresamos al factorial anterior
  • $n!=n\cdot (n-1)!$, donde, para este punto, el valor de $(n-1)!$ ya es conocido por lo cual habremos acabado pues, con lo anterior, ya podremos conocer el valor de $n!$.

En otras palabras, la funcin recursiva que buscamos implementar calcular el factorial de un nmero $n$, intentando calcular primero $(n-1)!,(n-2)!,(n-3)!,(n-4)!,\ldots,4!,3!,2!$. Como an no puede calcular $(n-1)!$ pasa a intentar calcular $(n-2)!$; como an no puede calcular $(n-2)!$ pasa a intentar calcular $(n-3)!$, y as sucesivamente hasta que logr calcular $2!=2$. Una vez que logramos calcular un valor fijo de un factorial, la funcin regresar a calcular los factoriales que quedaron pendientes. Esto es, ahora la funcin calcular $3!, 4!,\ldots, (n-3)!, (n-2)!,(n-1)!,n!$, pues, como ya conoce el valor de $2!$, ya puede calcular el valor de $3!$; como ya conoce el valor de $3!$ ya puede calcular el valor de $4!$ y as hasta llegar al valor de $(n-2)!$; como ya conoce el valor de $(n-2)!$ ya puede calcular el valor de $(n-1)!$ y, dado que ya conocemos este valor, finalmente podemos calcular el valor de $n!$.

def factorial(n) -> int:
    """Funcin de un parmetro que retorna un nmero
    entero. Esta funcin calcula el factorial de un
    nmero n dado."""
    # Esta variable almacenara el valor de las multiplicaciones
    # n(n-1)!. Lo inicializamos en uno
    fac_num = 1
    # Nuestra funcion solo actuara cuando el valor de
    # la n sea mayor a cero
    if n > 0:
        # implementamos la multiplicacion n(n-1)!
        # como explicamos antes. Esto es
        # n(n-1)! --> n * factorial(n-1)
        fac_num = n * factorial(n-1)
    else:
    # Cuando la n sea menor a 1 nuestra funcion finaliza
        print("fin")
    # Retornamos el valor final del factorial calculado
    return fac_num

def factorial2(n):
    """Funcin de un parmetro que retorna un nmero
