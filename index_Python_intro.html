Curso de Introducción a la Programación con Python




Descripción: Este curso tiene por objetivo introducir a los estudiantes en el mundo de la programación utilizando como herramienta principal el lenguaje Python. Asimismo, dentro de este curso buscamos introducir las herramientas básicas para el análisis de datos, utilizando para ello las librerías Pandas, NumPy y Matplotlib. Adicionalmente, al final del curso se brinda una introducción al lenguaje SQL utilizando la librería SQLite de Python En total, el curso consta de más de 20 horas de clases en vivo, 8 tareas, 8 problemas resueltos paso a paso y 8 prácticas guiadas. También, pondremos en práctica todo lo aprendido con dos proyectos. Si deseas consultar el temario con mayor detalle Pulsa aquí

Competencia

Descipción: a lo largo del curso se subirán 8 problemas de programación, los cuales requerirán muy pocos conocimientos de programación en Python, sin embargo requerirán ciertas habilidades de pensamiento lógico y de construcción de algoritmos. De tal manera, con lo aprendido respectivamente en el curso, tendrán las herramientas para poder resolverlos; asimismo, se darán algunas recomendaciones para resolver problemas de este tipo. Además, cada problema tendrá pistas para su resolución.
Dinámica: Se subirán dos problemas por semana hasta completar 8 problemas. Se calificará el código solución de cada problema y se colocará un puntaje del 1 al 10, dependiendo la forma en que dicho problema fue resuelto.




Premio: El estudiante que obtenga más puntos al final del curso ganará un pase doble para cualquier curso posterior que ofrezcamos, es decir, obtendrá de manera gratuita dos lugares para cualquier curso posterior que impartamos.
